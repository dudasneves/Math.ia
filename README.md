#  Math.ia — Interface de Inteligência Artificial para Matemática

O **Math.ia** é um projeto de interface conversacional (Web UI) inspirado em plataformas como o ChatGPT, porém projetado especificamente para a **resolução, aprendizado e suporte em conceitos matemáticos**. 

>  **Fase Atual do Projeto:** Protótipo Front-End / Design de Interface. A estrutura visual e a experiência do usuário (UX/UI) estão desenvolvidas, prontas para futuras integrações com APIs de LLMs e engines de computação simbólica.

---

### Conceito & Proposta

A proposta do Math.ia é oferecer um ambiente limpo e intuitivo para estudantes e entusiastas de matemática interagirem com modelos de IA. A interface foi planejada para suportar:

*  **Chat Interativo:** Layout estilo conversacional focado em perguntas e respostas.
*  **Renderização de Fórmulas:** Estruturado visualmente para exibição fluida de notações matemáticas e equações.
*  **Design Responsivo & Clean:** Foco em usabilidade sem distrações visuais.

---

###  Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da aplicação web.
* **CSS3:** Estilização moderna, layout responsivo e design focado em UX.
* **JavaScript:** Manipulação de elementos da interface e comportamento dos componentes de chat.
